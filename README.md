# Unlink Post Terms

A WordPress plugin that adds an "Unlink Terms" toggle to the core Post Terms block. When enabled, term links are replaced with plain text, preserving all existing block styling options.

## Requirements

- WordPress 6.6+
- PHP 7.2+

## Installation

1. Upload the plugin files to `/wp-content/plugins/unlink-post-terms`
2. Activate the plugin through the Plugins screen in WordPress
3. Add a Post Terms block to any post or page — an "Unlink Terms" toggle will appear in the block settings panel
